# Command Plugins

Organize your commands into these categories. The loader will automatically scan these subfolders.

- **admin**: Group administrative tools
- **owner**: Bot owner exclusive commands
- **sudo**: Elevated administrative bypasses
- **economy**: Banking, XP, and gambling
- **ai**: Artificial Intelligence integrations
- **security**: Warden protection controls
- **downloaders**: Media retrieval tools
- **plugins**: Plugin marketplace management
